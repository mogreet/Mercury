NAME 

 Getopt - Contains the response from a getopt request to the Moms API

SYNOPSIS

use Getopt;
new Getopt($xmlDoc); #$xmlDoc is an xml file

DESCRIPTION

The Getopt object contains the response from a getopt request to the Moms API.

CONSTRUCTOR

Getopt

Getopt($xmlDoc) 
 This function creates a new Getopt response object.

Paramters: $xmlDoc: XML document that is returned by all the requests.

FUNCTIONS

getCampaignsIdList

getCampaignsIdList()

Returns the list of all the campaigns ID.

getCampaignStatusCode

getCampaignStatusCode($campaignId)

Returns the status code of the specified campaign. 0 if the specified campaign ID does not exist.

getCampaignStatus

getCampaignStatus($campaignId)

Returns the status label of the specified campaign. 0 if the specified campaign ID does not exist.

printElts

printElts()

Prints each campaign with its id, status code, content.
