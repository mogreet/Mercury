NAME 

 Uncache - Contains the response from a uncache request to the Moms API

SYNOPSIS

use Uncache;
new Uncache($xmlDoc); #$xmlDoc is an xml file

DESCRIPTION

The Uncache object contains the response from a uncache request to the Moms API.

CONSTRUCTOR

Uncache

Uncache($xmlDoc) 
 Creates a new Uncache response object.

Paramters: $xmlDoc: XML document that is returned by all the requests.

FUNCTIONS

getNumber

getNumber()

Returns the number of the uncached phone.

printElts

printElts()

Prints all the elements.
