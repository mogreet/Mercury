NAME 

 Lookup - Contains the response from a lookup request to the Moms API

SYNOPSIS

use Lookup;
new Lookup($xmlDoc); #$xmlDoc is an xml file

DESCRIPTION

The Lookup object contains the response from a lookup request to the Moms API.

CONSTRUCTOR

Lookup

Lookup($xmlDoc) 
 Creates a new Lookup response object.

Paramters: $xmlDoc: XML document that is returned by all the requests.

FUNCTIONS

getCampaignId

getCampaignId()

Returns the campaign ID of the requested transaction.

getFromNumber

getFromNumber()

Returns the sender number of the requested transaction.

getFromName

getFromName()

Returns the sender name of the requested transaction.

getToNumber

getToNumber()

Returns the receiver number of the requested transaction.

getToName

getToName()

Returns the receiver name of the requested transaction.

getContentId

getContentId()

Returns the content ID of the requested transaction.

getStatus

getStatus()

Returns the status label of the requested transaction.

getTimestampsList

getTimestampsList()

Returns the list of all the timestamps events of the requested transaction.

getEventList

getEventList()

Returns the list of all the events of the requested transaction.

getEvents

getEvents($timestamp)

Returns the list of the events that occured at the specified timestamp of the requested transaction. Null if the specified timestamp does not exist.

getEventsTC

getEventsTC()

Prints all the events with their timestamp and content.

printElts

printElts()

Prints all the elements.
