NAME 

 Setopt - Contains the response from a setopt request to the Moms API

SYNOPSIS

use Setopt;
new Setopt($xmlDoc); #$xmlDoc is an xml file

DESCRIPTION

The Setopt object contains the response from a setopt request to the Moms API.

CONSTRUCTOR

Setopt

Setopt 
 Creates a new Setopt response object.

Paramters: $xmlDoc: XML document that is returned by all the requests.
