NAME 

 Ping - Contains the response from a ping request

SYNOPSIS

use Ping;
new Ping($xmlDoc); #$xmlDoc is an xml file

DESCRIPTION

The Ping object contains the response from a ping request to the Moms API.

CONSTRUCTOR

Ping

Ping($xmlDoc) 
  Creates a new Ping response object.

Parameters: $xmlDoc : XML document that is returned by all the requests.
