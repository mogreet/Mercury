NAME 

 Transactions - Contains the response from a transactions request to the Moms API

SYNOPSIS

use Transactions;
new Transactions($xmlDoc); #$xmlDoc is an xml file

DESCRIPTION

The Transactions object contains the response from a transactions request to the Moms API.

CONSTRUCTOR

Transactions

Transactions($xmlDoc) 
 Creates a new Transactions response object.

Paramters: $xmlDoc: XML document that is returned by all the requests.

FUNCTIONS

getCampaignsIdList

getCampaignsIdList()

Returns the list of all the campaigns ID.

getCampaignNames

getCampaignNames($campaignId)

Returns the names for the specified campaign ID (a campaign can have more than one name).

getTransactionsIdList

getTransactionsIdList()

Returns the list of all the transactions ID.

getTransactionsIdListFrom

getTransactionsIdListFrom($campaignId)

Returns the list of all the transactions ID for the specified campaign ID.

getValue

getValue($campaignId,$transactionId,$key)

Returns the value for the specified key.

getHash

getHash($campaignId,$transactionId)

Returns the hash corresponding to the specified campaign ID and transaction ID. Null if the campaign ID or the transaction ID does not exist.

getDatestamp

getDatestamp($campaignId,$transactionId)

Returns the datestamp corresponding to the specified campaign ID and transaction ID. Null if the campaign ID or the transaction ID does not exist.

getFromNumber

getFromNumber($campaignId,$transactionId)

Returns the sender's number corresponding to the specified campaign ID and transaction ID. Null if the campaign ID or the transaction ID does not exist.

getFromName

getFromName($campaignId,$transactionId)

Returns the sender's name corresponding to the specified campaign ID and transaction ID. Null if the campaign ID or the transaction ID does not exist.

getToNumber

getToNumber($campaignId,$transactionId)

Returns the receiver's number corresponding to the specified campaign ID and transaction ID. Null if the campaign ID or the transaction ID does not exist.

getToName

getToName($campaignId,$transactionId)

Returns the receiver's name corresponding to the specified campaign ID and transaction ID. Null if the campaign ID or the transaction ID does not exist.

printElts

printElts()

Prints each campaign with its informations.
