NAME 

 Info - Contains the response from an info request

SYNOPSIS

use Info;
new Info($xmlDoc); #$xmlDoc is an xml file

DESCRIPTION

The Info object contains the response from a info request to the Moms API.

CONSTRUCTOR

Info

Info($xmlDoc) 
 This function creates a new Info response object.

Paramters: $xmlDoc: XML document that is returned by all the requests.

FUNCTIONS
