NAME 

 Send - Contains the response from a send request to the Moms API

SYNOPSIS

use Send;
new Send($xmlDoc); #$xmlDoc is an xml file

DESCRIPTION

The Send object contains the response from a send request to the Moms API.

CONSTRUCTOR

Send

Send($xmlDoc) 
 Creates a new Send response object.

Paramters: $xmlDoc: XML document that is returned by all the requests.

FUNCTIONS

getMessageID

getMessageID()

Returns the message ID.

getHash()

Returns hash.
